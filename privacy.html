Privacy Policy

Last updated: February 2026

This application is a personal automation tool used to upload videos to social media platforms including TikTok via their official APIs.

Information We Collect

This application does not collect, store, or share any personal data from end users. The application operates solely to automate video posting to social media platforms on behalf of the account owner.

How We Use Information

No personal information is collected or used. OAuth tokens are stored locally and used only to authenticate with social media APIs.

Third Party Services

This application uses the following third party APIs:

  • TikTok API — for video uploads
  • YouTube Data API — for video uploads

Data Security

All authentication credentials are stored locally on the operator's private server and are never transmitted to third parties.
